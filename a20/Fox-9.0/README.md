# OrangeFox R11 A20 - a20

## How to build

```bash
# download script
$ wget https://raw.githubusercontent.com/yukosky/RecoverBuild/master/a20/Fox-9.0/build.sh
$ wget https://raw.githubusercontent.com/yukosky/RecoverBuild/master/a20/Fox-9.0/initial.sh

# Change the script permission
$ chmod +x ./build.sh
$ chmod +x ./initial.sh

# Build & Sync
$ ./initial.sh; ./build.sh
