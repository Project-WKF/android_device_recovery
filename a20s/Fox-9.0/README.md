# OrangeFox R11 A20s - a20s

## How to build

```bash
# download script
$ wget https://raw.githubusercontent.com/Project-WKF/android_device_recovery/master/a20s/Fox-9.0/build.sh
$ wget https://raw.githubusercontent.com/Project-WKF/android_device_recovery/master/a20s/Fox-9.0/initial.sh

# Change the script permission
$ chmod +x ./build.sh
$ chmod +x ./initial.sh

# Build & Sync
$ ./initial.sh; ./build.sh
