# TWRP 10 Note 10 Lite - r7

## How to build

```bash
# download script
$ wget https://raw.githubusercontent.com/Project-WKF/android_device_recovery/master/r7/TWRP-10/build.sh
$ wget https://raw.githubusercontent.com/Project-WKF/android_device_recovery/master/r7/TWRP-10/initial.sh

# Change the script permission
$ chmod +x ./build.sh
$ chmod +x ./initial.sh

# Build & Sync
$ ./initial.sh; ./build.sh
